About ReleaseScraper

ReleaseScraper is a website for scraping publicly available information about major media releases and gathering it in one place.

Information is gathered from publishers' websites using Selenium and Python's BeautifulSoup and Requests libraries.

Once it's gathered, we display it here, for you!

This website also runs on Python- the Flask library lets us run html/css in Python!
